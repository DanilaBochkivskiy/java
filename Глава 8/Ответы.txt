2. Как реализован класс String, какие поля в нем объявлены?

В Java класс String устроен максимально эффективно. Основное хранилище — это массив байтов byte[] value, который хранит символы строки. Чтобы экономить память, используется поле byte coder: оно указывает, сжата ли строка или использует полную кодировку (UTF-16). Также внутри есть поле int hash, где сохраняется вычисленный хэш-код, чтобы не пересчитывать его каждый раз при использовании строки в коллекциях. Все эти поля скрыты и помечены как final, что делает строку неизменяемой и безопасной для работы в разных частях программы.
