2. Как определить, какие поля и методы необходимо определить в классе? 
Привести пример. Какие спецификаторы можно использовать с полями, 
а какие с методами (и что они значат)? 

Поля — это характеристики объекта (данные), методы — это действия объекта (поведение).

Спецификаторы:
Для полей: public, protected, private (доступ), static (общее для всех), final (константа).

Для методов: public, protected, private (доступ), static (вызов без объекта), abstract (без реализации), final (запрет переопределения).
